\((1)\qquad\qquad\qquad f(x)=0, \quad x\in [a,b],\)

де \(f:[a,b]\to \mathbb{R}\) -- неперервна на \([a,b]\) функція.

\emph{Чисельне розв'язування} такого рівняння cкладається з двох етапів:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  локалізація коренів, тобто пошук числових проміжків, на яких є тільки
  один корінь цього рівняння;
\item
  обчислення коренів з наперед заданою точністю на кожному з виділених
  проміжків.
\end{enumerate}

\subsection{\#\#\# 3.1.1.2 Графічна локалізація коренів
рівняння}\label{ux433ux440ux430ux444ux456ux447ux43dux430-ux43bux43eux43aux430ux43bux456ux437ux430ux446ux456ux44f-ux43aux43eux440ux435ux43dux456ux432-ux440ux456ux432ux43dux44fux43dux43dux44f}}

Для побудови графіка функції \texttt{f} скористаємося бібліотекою
\texttt{matplotlib}. Найпростіше це виконати в інтерактивному режимі,
який встановлють командою \texttt{\%matplotlib\ widget}. У цьому випадку
після виконання функції \texttt{plot\_graphics}, яка побудує потрібний
графік, треба виконати такі дії: 1. активізувати режим
\texttt{Zoom\ to\ rectangle} за допомогою іконки меню на графічній
панелі (яка містить зображення прямокутника) 2. на графічній панелі
визначити мишкою якомога меншу прямокутну область, яка містить точку
перетину графіка функції з віссю абсцис.

Виконання пункту 2 автоматично приведе до відображення визначеної
прямокутної області на всю графічну панель. Виконуючи цей пункт кілька
разів, можна локалізувати нуль функції на відрізку \texttt{{[}a,\ b{]}}
з потрібною точністю на тій самій панелі.

Якщо режим \texttt{widget} не встановлено, то функцію
\texttt{plot\_graphics} виконують кілька разів, послідовно уточнюючи
координати відрізку \texttt{{[}a,\ b{]}}поки не локалізують корінь з
потрібною точністю.

\begin{itemize}

\item
  Підготувати потрібні функції :

  \begin{enumerate}
  \def\labelenumi{\arabic{enumi}.}
  \tightlist
  \item
    виконати комірку, де виконується підготовка середовища (в JupyterLab
    розкоментувати рядок з \texttt{\%matplotlib\ widget}, щоб мати змогу
    масштабувати графік при локалізації у п.4 )
  \item
    виконати комірку з визначенням функції \texttt{plot\_graphics}
  \item
    виконати комірку з визначенням функції \texttt{f} (цю комірку
    виконувати кожного разу після внесення змін у визначення функції
    \texttt{f} )
  \end{enumerate}
\item
  Локалізувати потрібний корінь рівняння :

  \begin{enumerate}
  \def\labelenumi{\arabic{enumi}.}
  \tightlist
  \item
    виконати комірку, в якій задається відрізок
  \item
    виконати комірку з викликом функції \texttt{plot\_graphics} для
    побудови графіка
  \item
    уточнити (звузити) відрізок \texttt{{[}a,\ b{]}}, щоб на ньому
    знаходився лише один корінь рівняння
  \item
    якщо в середовищі доступний інтерактивний режим (наприклад, в
    JupyterLab), то активізувати \texttt{Zoom\ to\ rectangle} ( вибрати
    в меню на графічній панелі іконку з прямокутником) і виділити мишкою
    прямокутну область з потрібним фрагментом графіка
  \item
    пункти 1 - 3 можна повторити для точнішої локалізації потрібного
    кореня
  \end{enumerate}
\end{itemize}
